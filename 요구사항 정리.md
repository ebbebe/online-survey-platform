## 핵심 기능

### 1. 5점 선택 제약 시스템
- 각 섹션별 5점 선택 최대 3개 제한
- 5점 3개 선택 완료 시:
  - 나머지 문항 5점 선택 비활성화
  - 1~4점만 선택 가능

### 2. 설문 구성
**기본 정보**
- 주관식(4): 이름, 부서, 연령, 총 직장 경력기간
- 객관식(N): 성별, 학력, 직급 등

**척도 문항**
- 50문항
- 1점(전혀 그렇지 않다) ~ 5점(항상 그렇다)
- 섹션으로 그룹핑

### 3. 관리 시스템
**설문 관리**
- **생성**: 제목, 설명, 섹션, 문항 자유롭게 구성
- **수정**: 응답이 없는 설문만 수정 가능
  - 수정 범위: 제목, 설명, 섹션, 문항 전체
  - ID 유지: 기존 섹션/문항 ID 보존으로 데이터 정합성 유지
  - 응답 1개 이상 시: 수정 버튼 비활성화 + 차단
- **삭제**: 관리자 권한으로 삭제 가능 (응답 포함 전체 삭제)
- 각 설문 고유 URL 생성 (`/surveys/[id]`)
- 설문별 데이터 독립 관리

**데이터 구조**
- 교육회차별 설문 분리 (11월용, 12월용 등)
- 각 설문의 응답 데이터 독립 저장

### 4. 결과 조회
**조회 페이지**
- 실시간 업데이트
- 설문별 결과 페이지

**표시 항목**
- 총 응답 수
- 문항별 통계:
  - 주관식: 텍스트 목록
  - 객관식: 막대그래프 + 비율
  - 척도: 점수별 분포 + 평균
- 개별 응답 상세 보기
- 엑셀/CSV 다운로드
